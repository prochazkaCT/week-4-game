Snowflake Collecter!

Game Instructions:

  1. You are given a random number at the start of each game.
  2. There are four snowflakes below and each of them has been given a random value. By clicking on each snowflake, you will add that assigned value to your total score.
  3. You win the game by matching your total score to the randm number, you lose the game if your total score goes above the random number.
  4. The value of each item is hidden from you until you click on it.
  5. Each time when the game starts, the game will change the values of each item.
