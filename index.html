Welcome to the Snowflake Collecter!

Game Instructions:

  1. 1. You are given a random number at the start of each game (see the Number to Guess).
  2. 2. Each snowflake below has been assigned a random value.
  3. 3. When you click on each snowflake, your score will increase with each snowflake that you click.
  4. 4. You will need to figure out (quickly!) the value each snowflake has been given in order to match the random number to guess.
  5. 5. The game is won when you exactly match your score to the "Number to Guess".
  6. 6. Conversely, you lose the game if your total score goes above the random number.
  7. 7. Each time the game starts, the values of each snowflake will change.

    8. Good Luck!!!
